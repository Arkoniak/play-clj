(position! object x-val y-val)
(position! object x-val y-val z-val)

Sets the position of object. If object is a screen, the position of the camera will be set.

Source
(defn position!
  ([object vec-3]
    (position! object (x vec-3) (y vec-3) (z vec-3)))
  ([object x-val y-val]
    (position! object x-val y-val nil))
  ([object x-val y-val z-val]
    (when x-val (x! object x-val))
    (when y-val (y! object y-val))
    (when z-val (z! object z-val))))
