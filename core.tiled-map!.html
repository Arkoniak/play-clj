(tiled-map! screen k & options)

Calls a single method on a tiled-map.

(tiled-map! screen :get-layers)
Options
:dispose
:get-tile-sets
:set-owned-resources^com.badlogic.gdx.utils.Array resources
Used by loaders to set resources when loading the map directly, without {@link AssetManager}. To be disposed in {@link #dispose()}.
Source
(defmacro tiled-map!
  [screen k & options]
  `(let [^BatchTiledMapRenderer renderer# (u/get-obj ~screen :renderer)]
     (u/call! ^TiledMap (.getMap renderer#) ~k ~@options)))
