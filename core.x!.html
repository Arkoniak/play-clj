(x! object x-val)

Sets only the x position of object. If object is a screen, the x position of the camera will be set.

Source
(defn x!
  [object x-val]
  (cond
    (instance? Vector2 object) (let [^Vector2 v object]
                                   (.set v x-val (. v y)))
    (instance? Vector3 object) (let [^Vector3 v object]
                                   (.set v x-val (. v y) (. v z)))
    :else (let [^Camera camera (u/get-obj object :camera)]
            (set! (. (. camera position) x) x-val)
            (.update camera))))
