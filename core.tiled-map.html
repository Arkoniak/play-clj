(tiled-map s & options)

Returns a TiledMap. Normally, you don't need to use this directly.

Options
:dispose
:get-tile-sets
:set-owned-resources^com.badlogic.gdx.utils.Array resources
Used by loaders to set resources when loading the map directly, without {@link AssetManager}. To be disposed in {@link #dispose()}.
Source
(defn tiled-map*
  ([]
    (TiledMap.))
  ([^String path]
    (or (u/load-asset path TiledMap)
        (.load (TmxMapLoader.) path))))
(defmacro tiled-map
  [s & options]
  `(u/calls! ^TiledMap (tiled-map* ~s) ~@options))
